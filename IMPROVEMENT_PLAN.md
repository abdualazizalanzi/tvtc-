# تحسينات الإرشاد المهني والسيرة الذاتية

## الملف الأول: career-guidance.tsx (صفحة الإرشاد المهني)

### التحسينات المطلوبة:
1. **تحسين خوارزمية التحليل**: الخوارزمية الحالية بسيطة جداً وتعتمد على عد النقاط
2. **إضافة المزيد من الأسئلة**:增加到 10-12 سؤال لأداء أفضل
3. **إضافة تحليل مهارات**: تحليل الفجوات في المهارات وتقديم توصيات
4. **إضافة مسارات مهنية أكثر تفصيلاً**: تقديم مسارات مهنية بديلة
5. **إضافة نصائح للتطوير**: نصائح لتطوير المهارات المطلوبة

### التغييرات المطلوبة:
- توسيع قاعدة البيانات majorsDatabase بمزيد من التفاصيل
- تحسين دالة analyzeAnswersLocally لتأخذ جميع الإجابات بعين الاعتبار
- إضافة قسم "لماذا هذا التخصص مناسب لك" مع تفاصيل أكثر
- إضافة قسم "كيف تستعد لهذا المسار المهني"

---

## الملف الثاني: cv-generator.tsx (مولد السيرة الذاتية)

### التحسينات المطلوبة:
1. **تحسين جودة PDF**: تحسين إعدادات html2canvas
2. **إضافة صور**: رفع صورة شخصية
3. **تحسين القالب**: جعل السيرة الذاتية أكثر احترافية
4. **إضافة أقسام جديدة**: 
   - المشروعات الشخصية
   - المهارات التقنية مع مستوى
   - المراجع
5. **حل مشاكل الحفظ**: ربط زر الحفظ بـ API

### التغييرات المطلوبة:
- إضافة ميزة رفع الصورة الشخصية
- تحسين تنسيق PDF
- إضافة API لحفظ البيانات المعدلة
- إضافة المزيد من أنماط السيرة الذاتية

---

## ملفات API المطلوبة:
- `/api/profile` - GET/PUT للتعامل مع الملف الشخصي
- `/api/career/analyze` - لتحليل الإرشاد المهني (اختياري)
